Использование:
	python3 main.py [options]

Структура:
	- console.py - парсер и обработик аргументов консоли, для того, чтобы узнать все опции, необходимо запустить программу с ключом -h или --help
	- Constants.py - содержит константы, например, коды возврата, также содержит объект логирования
	- logger.py - это файл для логирования в файл app.log, на данный момент логиурет общение SMTP-клиента и SMTP-сервера, также некоторые Exception
	- mail.py - файл, который содержит в себе механизм формирования письма для отправки ее на сервер
	- main.py - основной файл, обеспечивает работу с конолью и отправляет письмо с вложениями на сервер
	- MIME.py - файл, содержащий почтовый стандарт MIME
	- MIME_tests.py - содержит модульные тесты для модуля MIME
	- MIMEException.py - содержит пользовательские Exception, созданные для потчового стандарта MIME
	- SMTP.py - SMTP-клиент для осуществления передачи электронной почты
	- SMTP_test.py - модульные тесты для SMTP.py 
	- SMTPExceptoin.py - пользовтельские Exception для клиента SMTP
	- types_map.json - файл, в котором хранится соответсвие между расширением  программы и типом контента(Content-type)
	- encodings_map.json - файл, в котором хранится соответсвие между расширением СЖАТОЙ программы и типом контента(Content-type)
	- app.log - содержит логи к программе
	- main_test.py - модульные тесты для main.py
	- dump.pickle - хранит последнее не отправленное сообщение


!! Так как в Линукс нет расширения, то будет отправлять с заголовком Content-Type: application/octet-stream
!! Работает только на питоне с 3 версии
